### Linu文件目录及其作用

| 目录  | 作用                                                     |
| ----- | -------------------------------------------------------- |
| /     | 根目录                                                   |
| /bin  | 主要放基本执行命令                                       |
| /sbin | 主要放系统系统管理命令                                   |
| /boot | 引导目录（与系统启动有关），放置内核、grub、initrd等文件 |
| /dev  | device，设备文件目录，也是特殊文件目录                   |
| /etc  | 配置文件目录，放置系统的配置文件                         |
